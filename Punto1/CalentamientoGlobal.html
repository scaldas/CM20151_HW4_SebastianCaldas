Calentamiento Global?¶

La idea de esta discusion es corroborar o cuestionar la nocion que se tiene que la actividad del hombre ha llevado a un aumento en la temperatura promedio de nuestra planeta. En este caso particular, se estudian datos para 5 ciudades Colombianas: Bogota, Cali, Barranquilla, Bucaramanga e Ipiales. Se pretende usar herramientas estadisticas basicas para lograr lo anterior.

Se comienza entonces definiendo una funcion que, para cada ciudad, nos muestra tanto una interpolacion por splines de grado 1 como una regresion lineal hasta tercer orden, ambas extrapoladas al año 2017. import csv
from scipy.interpolate import interp1d, splrep, splev 

def print_interpolations(city):
    ff = open('temperaturas.csv', 'rt')
    reader = csv.reader(ff)

    dates = []
    hash_temperaturas = {}
    
    for row in reader:
        if row[4] == city and float(row[3]) != 999.9:
            date = datetime.datetime.strptime(row[5],"%Y-%m-%d")
            dates.append(date)
            hash_temperaturas[date] = float(row[3])  

    dates = np.sort(dates)

    temperaturas = []
    for each in dates:
        temperaturas.append(hash_temperaturas[each])

    dates = matplotlib.dates.date2num(dates)
    
    max_date = datetime.datetime.strptime('2017-12-31',"%Y-%m-%d")
    max_date = matplotlib.dates.date2num(max_date)
    
    x_l = np.linspace(dates[0],max_date,10000)
    tck = splrep(dates, temperaturas, k=1)
    y_spl = splev(x_l, tck)
    
    regression = polyfit(dates, temperaturas, 1) 
    y_reg = regression[0]*x_l + regression[1]
    
    figure(figsize=(8,6))
    plot_date(x_l, y_spl, '-b', c='k')
    plot_date(dates, temperaturas, 'o')
    title('Temperatura ' + city + ' - Interpolacion por splines')
    xlabel('Fecha')
    ylabel('Temperatura(ºC)')
    legend(['linear', 'data'])
    
    figure(figsize=(8,6))
    plot_date(x_l, y_reg, '-g', c='k')
    plot_date(dates, temperaturas, 'o')
    title('Temperatura ' + city + ' - Regresion lineal')
    xlabel('Fecha')
    ylabel('Temperatura(ºC)')
    legend(['linear', 'data'])
    
    print('Pendiente Regresion: ' + str(regression[0]))

Conclusiones¶

Antes de concluir, es importante anotar lo siguiente: No se encontro el R^2 para las anteriores regresiones pues evidentemente este dara bastante bajo. Esto pues la regresion se utilizo para descubrir una tendencia en los datos y no para ajustarlos a un modelo. El ajuste es evidentemente terrible en todos los casos. La tendencia, por otro lado, es clara: Las temperaturas han tendido al alza en los ultimos años.

Es cierto que en el periodo de tiempo trabajado, todas las ciudades han experimentado epocas con temperaturas mas bajas de las habituales. Pero esto no significa que la tendencia estadistica este errada. No porque esta mañana haya hecho sol significa que Bogota es ahora un paraiso tropical. Hay un comportamiento marcado que apunta hacia un aumento de las temperaturas durante las ultimas decadas, asi la causa no sea posible hallarla con este analisis. El globo se calienta.
